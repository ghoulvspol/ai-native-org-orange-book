## 这是什么

一本关于AI Native组织转型的深度技术手册。从认知到架构，从工作流到文化，覆盖企业AI Native转型的完整链路。

22个章节，6个Part + 附录。

## 适合谁

- CEO、CTO、CHRO：推动AI Native转型的决策者
- CAIO：首席智能官或同等角色
- 组织变革推动者：负责落地执行
- 产品经理：设计AI Native产品

## 核心概念

- **AI Native**：AI能力长在组织基因里，不是贴在表面
- **CAIO**：Chief AI Officer，首席智能官
- **AI联合舰队**：跨部门AI协作小组
- **能量金**：AI使用的正向激励系统
- **数据飞轮**：数据→模型→体验→用户的正循环
- **Agent制**：围绕Agent能力边界划分的组织结构
